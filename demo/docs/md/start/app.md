# 运行方式

## 一、bin.sh脚本方式

> 单应用启动方式，提供开发和部署相关的命令

- 启动开发
```
./bin.sh
```

- 查看log
```
./bin.sh log
```

- 重启应用
```
./bin.sh restart
```

- 停止应用
```
./bin.sh stop
```

- 更新npm
```
./bin.sh npm
```

- 上线部署
```
./bin.sh online
```

- 线上更新
```
./bin.sh update
```

- 启动或重启nginx
```
./bin.sh nginx
```
